# 🍕 Pizza Sales SQL & Power BI Dashboard

## 📌 Project Overview
This project focuses on analyzing pizza sales data using SQL Server and Power BI to generate meaningful business insights and interactive visual reports.

The dashboard helps analyze:
- Sales performance
- Customer ordering trends
- Best and worst-selling pizzas
- Revenue insights
- Order patterns
- Pizza category and size performance

---

# 🛠 Tools & Technologies Used

- Microsoft Power BI Desktop
- SQL Server Express
- SQL Server Management Studio (SSMS 22)
- Microsoft Excel 2021
- Power Query
- DAX

---

# 📂 Dataset Used

- pizza_sales.csv

The dataset contains pizza order details including:
- Order ID
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity
- Order Date
- Total Price

---

# 📊 KPI Requirements

The following KPIs were analyzed:

1. Total Revenue
2. Average Order Value
3. Total Pizzas Sold
4. Total Orders
5. Average Pizzas Per Order

# 📌 Business Insights

- Orders are highest on Fridays and Saturdays.
- July, January, and March recorded the highest order volumes.
- Classic pizza category contributes the highest sales.
- Large-size pizzas generate maximum revenue.
- Thai Chicken Pizza generates the highest revenue.
- Classic Deluxe Pizza has the highest quantity sold and total orders.
- Brie Carre Pizza is the worst-performing pizza in revenue and orders.
